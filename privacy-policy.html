Privacy Policy

This extension collects certain data for the purpose of interacting with APIs provided by AI Provider selected by user, default AI provider is OpenAI. We are committed to protecting your privacy and handling your data securely. Please review our privacy policy for more information on how we collect, use, and protect your data. if you would like to enquire about project, please feel free to reach out to us at aicodecompanion@gmail.com.

Types of Data Collected

Extension collect data in below different forms, please find data purpose and storage details for each type of data in below sections

  • Text entered by user in editor, if user select the code and prompt query for that code
  • Text entered by user in Text area in form of prompts
  • Text entered by user in extension settings
  • Text entered by user in command palette to run the available commands for extension
  • Data provided by Context Menu options provided by extension

Purpose of Data Collection

  • Data provided by any option like textarea, command palette, running commands via context menu will directly be sent to selected AI Provider using secured HTTPS protocol and we don't store any of such data except maintaining a history of chat messages for retaining chat context
  • Data provided by user in extension settings will only be used for API calls to form the required/optional parameters
  • Access Key provided by user via extension command will only be used for API calls to authenticate the user and will get saved in encrypted form using AES encryption algorithm.
    For enhanced security we don't store encrypted key in extension settings rather it is stored in VS code storage API. Users can also pick their on Encryption Key to further enhance the security for their access key.

Data Storage and Security

We store the data wrt to chat history in VS code storage API Storage of API Key is explained in above section

Data Sharing

We done share data to any third party except the selected AI Provider for API calls
